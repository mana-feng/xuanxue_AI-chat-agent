## 📝 使用说明

### 使用方法（单一 .env）

1. **前端**：编辑 `frontend/.env`，开发/生产按需修改值（Vite 会读取该文件）。
2. **后端**：编辑 `backend/.env`，开发/生产按需修改值（启动时固定读取该文件）。

## ⚠️ 重要提示

1. **环境变量命名**：
   - 前端环境变量必须以 `VITE_` 开头
   - 后端环境变量无特殊要求

2. **配置文件优先级**：
   - 前端：`.env.production` > `.env.development` > 默认值
   - 后端：`.env.production` / `.env.development` > `.env` > 默认值

3. **敏感信息**：
   - `.env` 文件包含敏感信息，不要提交到Git仓库
   - 生产环境密钥必须使用强随机字符串

4. **修改配置后**：
   - 后端：需要重启服务
   - 前端：需要重新构建（生产环境）或重启开发服务器（开发环境）
