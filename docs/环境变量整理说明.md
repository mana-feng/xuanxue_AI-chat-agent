## 🔧 配置加载机制

### 前端配置加载

Vite 会根据 `NODE_ENV` 自动加载对应的 `.env` 文件：
- `npm run dev` → 加载 `.env.development`
- `npm run build` → 加载 `.env.production`

### 后端配置加载

代码会根据 `NODE_ENV` 自动选择文件：
- `NODE_ENV=development` → 加载 `.env.development`
- `NODE_ENV=production` → 加载 `.env.production`
- 如果对应文件不存在，会尝试加载 `.env`（如果存在）

**加载位置**：
- `backend/config/index.js` (第26行)
- `backend/db.js` (第11-30行)
- `backend/app.js` (第41-48行)
